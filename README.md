##Using

[MkDocs](https://www.mkdocs.org/)

[Material for MkDocs](https://squidfunk.github.io/mkdocs-material/)

and custom theme elements

##Install

```
pip install mkdocs
pip install mkdocs-material
```

##Upgrade

```
pip install -U mkdocs
pip install -upgrade mkdocs-material
```
